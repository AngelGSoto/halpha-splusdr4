1 1. Definitions and Problem Setup

1.1 Hα Residual

\[ \Delta = (r - J0660)_{\text{obs}} - (r - J0660)_{\text{fit}}, \quad \text{where } (r - J0660)_{\text{fit}} = m(r - i) + b. \]

1.2 Goal

Compute the uncertainty in \(\Delta\): \[ \sigma_{\text{est}} = \sqrt{\text{Var}(\Delta)}. \]

2 2. Classical Method (Overestimation)

2.1 Formula

\[ \sigma_{\text{est, classical}}^2 = \sigma_s^2 + \sigma_{(r - J0660)}^2 + m^2 \sigma_{(r - i)}^2. \]

2.2 Step-by-Step Derivation

  1. Expand color errors:

\[

\begin{aligned} \sigma_{(r - J0660)}^2 &= \sigma_r^2 + \sigma_{J0660}^2, \\ \sigma_{(r - i)}^2 &= \sigma_r^2 + \sigma_i^2. \end{aligned}

\]

  1. Substitute into the formula:

\[ \sigma_{\text{est, classical}}^2 = \sigma_s^2 + (\sigma_r^2 + \sigma_{J0660}^2) + m^2 (\sigma_r^2 + \sigma_i^2). \]

  1. Identify overestimation:

\[ \sigma_{\text{est, classical}}^2 = \sigma_s^2 + \underbrace{(1 + m^2)\sigma_r^2}_{\text{Double-counting}} + \sigma_{J0660}^2 + m^2 \sigma_i^2. \]

2.3 Numerical Example (\(m = 0.5\)):

\[ \sigma_r = 0.03, \ \sigma_i = 0.04, \ \sigma_{J0660} = 0.02 \Rightarrow \sigma_{\text{est, classical}} = 0.0616 \ \text{(+26.5\% error)}. \]

3 3. Proposed Method (Your Paper)

3.1 Formula

\[ \sigma_{\text{est, proposed}}^2 = \sigma_s^2 + (1 - m)^2 \sigma_{(r - J0660)}^2 + m^2 \sigma_{(r - i)}^2. \]

3.2 Step-by-Step Derivation

  1. Expand color errors:

\[

\begin{aligned} (1 - m)^2 \sigma_{(r - J0660)}^2 &= (1 - m)^2 (\sigma_r^2 + \sigma_{J0660}^2), \\ m^2 \sigma_{(r - i)}^2 &= m^2 (\sigma_r^2 + \sigma_i^2). \end{aligned}

\]

  1. Substitute into the formula:

\[ \sigma_{\text{est, proposed}}^2 = \sigma_s^2 + (1 - m)^2 \sigma_r^2 + (1 - m)^2 \sigma_{J0660}^2 + m^2 \sigma_r^2 + m^2 \sigma_i^2. \]

  1. Group \(\sigma_r^2\) terms:

\[ \sigma_{\text{est, proposed}}^2 = \sigma_s^2 + \underbrace{[(1 - m)^2 + m^2] \sigma_r^2}_{\text{Apparent Extra Term}} + (1 - m)^2 \sigma_{J0660}^2 + m^2 \sigma_i^2. \]

4 4. Perfect Method (Ideal Error Propagation)

4.1 Formula

\[ \sigma_{\text{est, perfect}}^2 = \sigma_s^2 + (1 - m)^2 \sigma_r^2 + \sigma_{J0660}^2 + m^2 \sigma_i^2. \]

4.2 Step-by-Step Derivation

  1. Express \(\Delta\) in terms of individual bands:

\[ \Delta = (1 - m)r - J0660 + mi - b. \]

  1. Compute variance:

\[ \text{Var}(\Delta) = (1 - m)^2 \sigma_r^2 + \sigma_{J0660}^2 + m^2 \sigma_i^2. \]

  1. Add intrinsic scatter (\(\sigma_s^2\)):

\[ \sigma_{\text{est, perfect}}^2 = \sigma_s^2 + (1 - m)^2 \sigma_r^2 + \sigma_{J0660}^2 + m^2 \sigma_i^2. \]

5 5. Equivalence Under \(\sigma_{J0660}^2 \ll \sigma_r^2 + \sigma_i^2\)

5.1 Observational Justification

In surveys like SPLUS, the narrow \(J0660\) filter has lower noise: \[ \sigma_{J0660}^2 \ll \sigma_r^2 + \sigma_i^2 \quad \Rightarrow \quad (1 - m)^2 \sigma_{J0660}^2 \ll [(1 - m)^2 + m^2] \sigma_r^2 + m^2 \sigma_i^2. \]

5.2 Mathematical Equivalence

The difference between methods becomes negligible: \[ \Delta \sigma^2 = \sigma_{\text{est, proposed}}^2 - \sigma_{\text{est, perfect}}^2 = m^2 \sigma_r^2 - (2m - m^2)\sigma_{J0660}^2 \approx 0. \]

6 6. Numerical Validation (SPLUS DR4 Parameters)

6.1 Inputs

\[ m = 0.5, \quad \sigma_r = 0.03, \quad \sigma_i = 0.04, \quad \sigma_{J0660} = 0.02, \quad \sigma_s = 0. \]

6.2 Results

Method \(\sigma_{\text{est}}\) Error vs. Perfect
Classical 0.0616 +26.5%
Proposed 0.0308 -3.8%
Perfect 0.0320 0%

6.3 Interpretation

  • The proposed method’s error is minimal (<5%) and does not exclude true candidates (100% overlap with perfect method in your paper).
